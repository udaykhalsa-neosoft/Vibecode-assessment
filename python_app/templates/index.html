HealthCare Pro

Hospital Management System API

System Status: Active

Welcome to the HealthCare Pro backend service. This project is strictly an API-driven application designed to handle core hospital management operations securely and efficiently.

Available API Modules

  • Authentication: /login, /register, /auth/profile
  • Patient Management: /hospital/patients
  • Doctor Management: /hospital/doctors
  • Appointments: /hospital/appointments
  • Medical Records: /hospital/medical-records
  • Analytics & Config: /hospital/stats, /config

Note: Please refer to the provided Postman collection and project README.md for detailed endpoint documentation, required JSON payloads, and query parameter specifications.
